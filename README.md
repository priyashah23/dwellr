# dwellr
Make finding your home like finding your person - because it's personal. 

## RUNNING THE BACKEND 
1. `cd dwellr-backend`
2. `docker-compose up --build`
3. 3. Navigate to the '/' endpoint to migrate database

## RUNNING THE APP LOCALLY
1. `cd frontend`
2. `npm install`
3. `npm run dev`
4. `http://localhost:5173/`
